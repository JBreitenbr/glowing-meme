World Mosaic sammelt Tagesrhythmen aus aller Welt :

ruhige, klare Miniaturen aus Städten, Landschaften, Hügeln, Treppen, Gassen, geschrieben im Stil nüchterner, kleiner Porträts.
- Kein Drama.
- Keine Tourismusästhetik.
- Keine Überhöhung.
Nur Alltag als präzises kulturelles Mikroklima.

Jede Seite zeigt:
- eine Person
- einen Ort
- mehrere Zeitpunkte
- eine Atmosphäre
- ein reduziertes lokales Design (Farben, Texturen, Icons)

So entsteht ein globales Mosaik, gebaut aus den unspektakulären Momenten, die am ehrlichsten von einer Kultur erzählen.
